# SMV Client

Frontend for Smart Multi-Vendor E-Commerce Marketplace (React + Redux Toolkit + Tailwind + Vite).

## Run
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## Main Areas
- Customer: browse products, cart, orders
- Seller: dashboard, products, order status updates
- Admin: dashboard, users, sellers, categories, orders
