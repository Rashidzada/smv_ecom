# SMV Server

Backend API for Smart Multi-Vendor E-Commerce Marketplace (Node.js + Express + MongoDB).

## Run
```bash
npm install
cp .env.example .env
npm run seed
npm run dev
```

## Key Endpoints
- `/api/auth`
- `/api/categories`
- `/api/products`
- `/api/cart`
- `/api/orders`
- `/api/seller`
- `/api/admin`
